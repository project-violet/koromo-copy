## 유틸리티

### Zip Viewer ( https://github.com/dc-koromo/koromo-copy/releases/tag/zipviewer )

탐색기(Explorer)에서 폴더->오른쪽 클릭시 나타나는 메뉴에 `Zip Viewer`를 추가합니다.
이 프로그램은 선택한 폴더에 포함된 모든 `*.zip`파일을 가져오고, 작품의 정보와 이미지 한장을 추출해 보여줍니다.
